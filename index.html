Project Description

In this project we imagine that people now have access to a rapidly growing number of public data sources from the web/cloud which are getting augmented with personal sensors which now commonly talk to mobile phones or web pages. We look at designing an interface to make this kind of data and more available on your personal bathroom mirror (single person use), which could act as a large display touch display.

Project Sketches

Sketch 1

This is how the mirror starts and is at its default. It is complete with a Hide All button in the upper left corner, a light change button, a voice search button, and 5 "Lifestyle" buttons: Calendar, News and Messages, Clock, Weather, and Health (listed from top to bottom). A digital clock is also displayed in the upper right corner.

Sketch 2

A low priority feature, the voice search allows users to play music or ask basic questions. A voice wave appears between the speech and light button to indicate the mirror is waiting for input.

Sketch 3

This feature allows users to hide all buttons in order to prioritize mirror usage.

Sketch 4

This sketch demonstrates the usage of the calendar feature. The calendar also displays up to two upcoming events. A back button is included to re-minimize the calendar. This back button is included in all tabs.

Sketch 5

This sketch demonstrates the usage of the news feed feature which also includes messages and social media notifications. All tabs also include a pop-out feature in order to open a corresponding application on the user's phone.
